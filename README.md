## Rich Text Editor

The `rich_text_editor/` folder contains a standalone HTML/CSS/JS rich text editor that you can open locally.

### How to run locally

1. Open `rich_text_editor/index.html` in any modern web browser.
2. The editor runs entirely in the browser and saves drafts to `localStorage` automatically.

### Supported features

- Bold, italic, underline
- Font family and size selection
- Text color and highlight color
- Alignment controls (left, center, right, justify)
- Ordered and unordered lists, indent, outdent
- Undo/redo
- Insert link, image, and table
- Autosave to `localStorage`
- Download the editor content as HTML
